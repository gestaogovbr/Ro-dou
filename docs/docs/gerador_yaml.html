Monte sua busca no Diário Oficial

Preencha o formulário abaixo para criar o arquivo que faz o robô do Ro-DOU monitorar o Diário Oficial e avisar você por e-mail quando encontrar algo. Não é preciso saber programar — em poucos minutos o arquivo fica pronto.

Identificação da busca
dag

1. Identificação

Um apelido curto, sem espaços nem acentos, para você reconhecer esta busca depois.
Explique em uma frase o objetivo, para outras pessoas entenderem.
Com que frequência o robô repete a busca automaticamente.
Não existe opção "rodar só manualmente": se você não escolher um horário, o Ro-DOU roda a busca automaticamente todo dia de madrugada (o minuto exato varia por busca, para não sobrecarregar o sistema).
Nomes de quem cuida desta busca, separados por vírgula.
Ajudam a organizar as buscas no sistema, separadas por vírgula.
E-mails que recebem aviso caso o robô dê erro, separados por vírgula.
O que e onde monitorar
search

2. O que monitorar

Escreva uma por linha. O robô avisa você toda vez que qualquer uma delas aparecer no Diário. Obrigatório se a busca incluir o Querido Diário.
Você pode marcar mais de uma opção. Na dúvida, deixe apenas o DOU.

Você não precisa mexer aqui para a busca funcionar. Ajuste apenas se souber o que quer.

Quanto tempo para trás o robô procura a cada execução.
Marque as seções onde procurar. Se nenhuma for marcada, busca em todas.
Resultados que contenham estas palavras são descartados (uma por linha).
Uma unidade por linha, com o nome idêntico ao usado na publicação.
Um tipo por linha, escrito exatamente como no lista oficial de tipos de publicação, ex.: Portaria, Despacho, Edital, Ato, Resolução.

recursos do INLABS só valem se você marcou essa fonte acima.

Apenas INLABS. Padrão: 400 caracteres.

recursos do Querido Diário só valem se você marcou essa fonte acima.

Código IBGE do município (ex.: 4106902 = Curitiba/PR). Pode informar mais de um, separados por vírgula.

recursos do DOESP só valem se você marcou essa fonte acima.

Um caderno por linha, ex.: Executivo, Legislativo. Se nenhum for informado, busca em todos.
